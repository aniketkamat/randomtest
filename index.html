fastrerandomize

An R Package for Fast Rerandomization Using Accelerated Computing

fastrerandomize released!

2024-01-17

Tools

Employs JAX backend, batched processing, and GPU acceleration to allow for use in cases where the sample size or number of covariates is very large.

Improvements

Allows for substantial performance gains and use in cases where the number of covariates is very large.

Speed ups in applications to economic and geographical experiments.
